Jika ingin mendiagnosa lebih detail maka terdapat
button diagnosa. Jika button diagnosa di klik maka akan
tampil diagnosa detail dari setiap proses seperti yang
ditunjukkan pada gambar 4. 10. dimana proses segmentasi
image di tampilkan se cara rinci, seperti bentuk bintik adalah
gelendung, dimana terdapat tinggi objek, lebar objek dan
bentuk objek. Juga terdapat warna bintik dan warna
sempadan bintik. Lebih detail untuk warna sempadan juga
menampilkan warna disekeliling bintik. Warna daun j uga di
identifikasi pada window ini.
